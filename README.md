# 🖳 About 42 and this Repository

[42](https://42.fr/) is an innovative software engineering school with a focus on practical and collaborative learning (peer-to-peer), without teachers or traditional classes. The curriculum is gamified and project-based, challenging students to master everything from the fundamentals of C programming to complex architectures and web development.

This repository is a **monorepo** that consolidates all projects developed by me during the 42 Core Curriculum. It serves as a portfolio of my progression and technical evolution throughout each school "Milestone".
